## loading 组件

### type
```html
    <loading type="spinner"></loading>
    <loading type="circle"></loading>
```


### size 图标大小
```html
    <loading size="24px"></loading>
```


### text-size 文字大小
```html
    <loading text-size="14px"></loading>
```



### vertical 是否垂直排列文字
```html
    <loading vertical="true"></loading>
    <loading vertical="false"></loading>
```
